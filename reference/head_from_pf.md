# Convert pF to matric head (cm)

Convert pF to matric head (cm)

## Usage

``` r
head_from_pf(pf)
```

## Arguments

- pf:

  Numeric vector of pF values.

## Value

Numeric vector of matric head values in cm.

## Examples

``` r
head_from_pf(c(0, 1, 2, 3, 4.2))
#> [1]     1.00    10.00   100.00  1000.00 15848.93
```
